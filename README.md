# Arduino-robotic-arm
This project is to create a robotic arm controlled by an Arduino with 5 degrees of freedom.

### Components
- Arduino Uno
- 4 Servo Motors (MG996R)
- Servo Motor Driver (PCA9685)
- 4 Potentiometers (10K)
- Power Supply (10A)

Main body components are 3D printed. Files included.

### How it Works
A servo motor at each joint is controlled independently by its own potentiometer; a rotation of the potentiometer is equivalently followed by the motor.
